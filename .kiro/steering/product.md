# Product Overview

EOS UI Components is an Angular component library that provides reusable UI components for web applications. The library includes common interface elements like buttons, modals, cards, tables, tooltips, login forms, search components, and navigation elements.

## Key Features

- Standalone Angular components (no module dependencies)
- TypeScript support with proper type definitions
- Customizable variants and sizes for components
- Accessibility-focused design patterns
- Consistent styling and theming approach

## Target Use Cases

- Building consistent user interfaces across Angular applications
- Rapid prototyping with pre-built components
- Maintaining design system consistency
- Reducing development time with reusable components

The library is designed to be lightweight, tree-shakeable, and easy to integrate into existing Angular projects.
